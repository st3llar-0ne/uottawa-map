# uottawa-map

Utilisez la carte interactive ci-dessous pour explorer les bâtiments, les services, les trajets intérieurs et les arrêts de transport en commun. Cliquez sur un bâtiment pour accéder aux ressources disponibles, aux informations d'accessibilité et aux connexions intérieures. Basculez entre français et anglais à l'aide du bouton EN dans la carte.
